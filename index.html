Praktijk

  1. HTML-document

    Maak een HTML document aan met de volgende structuur:

    <!DOCTYPE html>[1]
    <html lang="nl">[2]
    <head>
     <title>[3]</title>
    </head>
    <body>
    </body>
    </html>
    • [1] We maken een HTML document.
    • [2] De inhoud van onze website is in het nederlands ('nl').
    • [3] Geef je website hier een titel.

  • Layout

    Maak in de body tag een layout met de volgende structuur:

    <header>(4)
    </header>

    <aside>(5)
    </aside>

    <main>(6)
    </main>

    <footer>(7)
    </footer>
    1. We maken een HTML document.
    2. De inhoud van onze website is in het nederlands ('nl').
    3. Geef je website een naam.
    4. In de header komt het logo.
    5. In de aside komt het menu.
    6. In de main komt de inhoud van onze website.
    7. In de footer komt de copyright.
